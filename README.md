# IV International Internet Olympiad in Informatics (HNGU) - Solutions 🏆

This repository contains comprehensive solutions for the technical and logical challenges presented during the **IV International Internet Olympiad in Informatics (HNGU 2026)**. The competition was organized by the Oguz Han University of Engineering and Technology of Turkmenistan.

## 🌟 Olympiad Tasks

The project provides optimized solutions for the following challenge categories:

- **📦 Annex 6 (Logistics Optimization):** Algorithms to optimize delivery routes while considering CO2 emission limits and strict time windows.
- **⚡ Annex 7 (Energy Management):** Systems to calculate and balance energy stability using solar power and battery storage constraints.
- **🦠 Virus Detection (Image Analysis):** Automated scanning of BMP image layers to identify specific virus color signatures.
- **👤 Forensic Data Extraction:** Heuristic and regex-based extraction of metadata, dates, and names from `.docx` documents and binary `.exe` files.
- **🔐 Cryptography & Sequences:** Solvers for Caesar ciphers and complex mathematical recurrences (e.g., `A26` sequence).

## 🛠 Technology Stack

- **Language:** Python 3.12+
- **Key Libraries:**
  - `python-docx` (Document parsing)
  - `Pillow` (Image processing)
  - `sympy` (Mathematical computations)
  - `subprocess` & `csv` (Task management and data handling)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BehruzMaxmudov1203/HNGU-International-Informatics-Olympiad.git
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Solver Hub:**
   Launch the central menu to access all solutions:
   ```bash
   python main.py
   ```
