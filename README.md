# IV International Internet Olympiad in Informatics (HNGU) - Solutions 🏆

This repository contains the official award-winning solutions for the **IV International Internet Olympiad in Informatics (HNGU 2026)**.

## 🥉 Achievement: 3rd Place (Bronze Medal)
I am honored to have achieved **3rd Place** in this prestigious international competition, competing against talented participants from countries including **China, Germany, Russia, Georgia, Kazakhstan, Kyrgyzstan, Turkmenistan, and Uzbekistan**.

The competition was organized by the Oguz Han University of Engineering and Technology of Turkmenistan.

---

## 🌟 Olympiad Tasks

The project provides optimized solutions for high-complexity challenge categories:

- **📦 Annex 6 (Logistics Optimization):** Advanced algorithms to optimize delivery routes under strict CO2 emission limits and time constraints.
- **⚡ Annex 7 (Energy Management):** Stability calculation systems balancing solar power output and battery storage efficiency.
- **🦠 Virus Detection (Image Analysis):** High-precision scanning of BMP image layers to identify specific virus color signatures.
- **👤 Forensic Data Extraction:** Heuristic-based data mining of metadata, dates, and names from `.docx` and binary `.exe` files.
- **🔐 Cryptography & Sequences:** Efficient solvers for Caesar ciphers and complex mathematical recurrences (e.g., `A26` sequence).

## 🛠 Technology Stack

- **Language:** Python 3.12+
- **Key Libraries:**
  - `python-docx` (Document parsing)
  - `Pillow` (Image processing)
  - `sympy` (Mathematical computations)
  - `subprocess` & `csv` (Task management and data handling)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BehruzMaxmudov1203/HNGU-International-Informatics-Olympiad.git
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Solver Hub:**
   Launch the central menu to access all solutions:
   ```bash
   python main.py
   ```
